### Project Setup
#### 1\. Create a Virtual Environment
```bash
python -m venv .venv
```

-----

#### 2\. Activate the Environment
```bash
.venv\Scripts\activate
```

-----

#### 3\. Configure API Key
```bash
copy .env.example .env
```

-----

#### 4\. Install Dependencies
```bash
pip install -r requirements.txt
```

-----

#### 5\. Run the Application
```bash
python main.py
```
